# Idea-Plugin-Cipher-tools
A simple encryption and decryption, encoding and decoding tool plugin. Support AES, DES, 3DES and so on

这是一个简单的加密解密、编码解码工具插件，可以安装到idea、android studio，只是做了简单的封装。

- MD5、SHA-1、SHA-256
- AES、RSA、DES、3DES
- base64
